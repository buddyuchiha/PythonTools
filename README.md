# Консольный файловый менеджер

Файловый менеджер, который позволяет пользователю выполнять базовые операции с файлами и директориями с помощью библиотеки os.

## Функционал

* Просмотр текущей директории
* Создание новой директории
* Удаление файла/директории
* Переименование файла/директории
* Копирование файла
* Перемещение между директориями
* Получение информации о файле (размер, дата создания и т. д.)
