# Log Validator & Parser

Утилита для валидации и анализа лог-файлов с использованием регулярных выражений

## Функционал

* Валидация формата строк в лог-файлах
* Анализ корректных записей и статистика
* Статистика по уровням логирования (INFO, WARNING, ERROR, DEBUG)
* Определение временного диапазона записей
* Поддержка различных форматов логов через конфигурацию
