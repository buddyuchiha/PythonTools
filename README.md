# Банковский счет с обработкой исключений

Класс BankAccount, имитирующий работу банковского счёта с комплексной системой проверок и пользовательскими исключениями

## Функционал

* Пополнение и снятие средств с валидацией операций
* Комплексная система пользовательских исключений
* Ведение истории транзакций
* Блокировка и разблокировка счета
* Сохранение истории операций в файл
* Логирование всех операций и ошибок
