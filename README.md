# Многопоточный сканер портов

Консольная утилита для многопоточной проверки доступности портов на заданном хосте

## Функционал

* Сканирование диапазона портов на указанном хосте
* Многопоточная обработка для ускорения сканирования
* Логирование результатов в файл
* Замер времени выполнения операций
* Обработка ошибок и исключений
