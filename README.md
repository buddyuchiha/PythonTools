# DataMaster: Утилита для работы с особыми типами данных

Консольная утилита для демонстрации работы с нестандартными типами данных в Python

## Функционал

* Работа с датой и временем: определение дня недели, разница в днях, конвертация временных зон
* Высокоточные вычисления с Decimal и сравнение с float
* Генерация UUID различных версий
* Кодирование строк в различные форматы
