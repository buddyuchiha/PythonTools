# Декоратор для логирования вызовов функций

Универсальный декоратор для автоматического логирования вызовов функций с применением к модулю имитации работы с API

## Функционал

* Автоматическое логирование начала и завершения выполнения функций
* Замер времени выполнения функций
* Логирование аргументов и возвращаемых значений
* Обработка и логирование исключений
* Настройка уровней логирования для разных обработчиков
