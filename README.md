<img width="272" height="89" alt="image" src="https://github.com/user-attachments/assets/a3f768dd-7885-442a-b993-02c7eab7ef65" /># Многопроцессорный анализатор файлов

Консольное приложение для параллельного анализа текстовых файлов с использованием многопроцессорной обработки

## Функционал

* Подсчет количества строк в текстовых файлах
* Подсчет количества слов (разделенных пробелами и переводами строк)
* Подсчет количества символов (включая пробелы и переносы строк)
* Параллельная обработка файлов в отдельных процессах
* Сбор общей статистики по всем обработанным файлам
