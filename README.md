# Конфигурационный менеджер и конвертер файлов

Утилита для работы с конфигурационными файлами разных форматов: чтение, валидация и конвертация между форматами

## Функционал

* Чтение и парсинг файлов различных форматов
* Конвертация между поддерживаемыми форматами
* Валидация данных и обработка ошибок
* Создание и удаление конфигурационных файлов
* Поддержка структурированных и неструктурированных данных
