Техническое задание для мини-проекта на Python: Многопоточный сканер портов
Цель проекта
Создать простой многопоточный сканер портов, который проверяет доступность указанных портов на заданном хосте.
Функциональные требования
Ввод данных:

Пользователь вводит:

IP-адрес или доменное имя (например, localhost, google.com).

Диапазон портов (например, 80-100).

Количество потоков (по умолчанию — 5).

Сканирование портов:

Каждый поток проверяет доступность порта (например, через socket).

Если порт открыт — выводить сообщение (например, Port 80 is open).

Если порт закрыт — ничего не выводить (или опционально в debug-режиме).

Синхронизация потоков:

Использовать Lock для безопасного вывода результатов.

Обеспечить корректное завершение всех потоков.

Обработка ошибок:

Обрабатывать возможные исключения (например, недоступность хоста, некорректный ввод).

Опционально (для усложнения):

Замер времени выполнения сканирования.

Сохранение результатов в файл.
