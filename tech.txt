Техническое задание для мини-проекта на Python
Тема: Работа с необычными типами данных в Python

Название проекта:
"DataMaster: Утилита для работы с особыми типами данных"

Цель:
Создать консольную утилиту, демонстрирующую работу с различными нестандартными типами данных в Python:

Дата и время (datetime, timezone, timedelta)

Высокоточные вычисления (Decimal)

Уникальные идентификаторы (UUID)

Кодировки строк (str.encode, bytes.decode, codecs)

Функциональные требования:
Работа с датой и временем

Принимает от пользователя дату в формате ДД.ММ.ГГГГ и выводит:

День недели для этой даты.

Разницу между этой датой и текущей в днях.

Пример использования временных зон (например, конвертация в UTC).

Работа с Decimal

Принимает два числа и операцию (+, -, *, /).

Выполняет вычисление с высокой точностью (без плавающей погрешности).

Сравнивает результат с обычным float и выводит разницу (если есть).

Генерация UUID

Генерирует UUID версий 1, 3, 4 и 5.

Показывает разницу между ними (например, на основе namespace).

Работа с кодировками

Принимает строку и кодирует её в UTF-8, CP1251, Base64.

Обрабатывает ошибки (например, неверную кодировку).

Нефункциональные требования:
Код должен быть чистым, с типами (TypeHints) и docstrings.

Использовать argparse или click для CLI.

Покрыть основные кейсы обработки ошибок (например, неверный формат даты).

Логирование действий (можно в stdout).
